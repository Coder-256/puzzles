Simon Tatham's Portable Puzzle Collection

This is a port of Simon Tatham's Portable Puzzle Collection to the iPhone and iPad.

Each puzzle game has its own help page describing the object of the game and how to play.

Touch Controls

The included instructions are the same as for all other versions. They refer to "left click" and "right click" for mouse control. To left click in this version, tap on a game tile. To right click, press and hold on a game tile. A short keyclick sound will confirm a right click.

About This Version

This version of Simon Tatham's Portable Puzzle Collection has been ported to iOS by Greg Hewgill. While every care has been taken to support all the features of each puzzle, you might still notice something is not quite right. Please let me know if you have any suggestions for improvement, or problem reports, at <greg@hewgill.com>.
